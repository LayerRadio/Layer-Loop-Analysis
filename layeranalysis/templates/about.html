Layer Loop Analysis

About

Layer Loop Analysis is a Dashboard for Loopring NFT Collections and Loopring Wallets (Wallet Analysis coming Soon)

  • This is the Alpha release of Layer Loop Analysis! Code for the LayerLoopAnalysis can be found at: https://github.com/LayerRadio/Layer-Loop-Analysis
  • Layer Loop Analysis retrieves the Transactions of a Collection's contract address and stores them locally to a SQL Database.
  • Loopring is an Ethereum Layer 2 zk-Rollup. Learn more about Loopring at loopring.io
  • Please let me know if you have any suggestions or features you would like. I would also appreciate suggestions and collaborations on code
